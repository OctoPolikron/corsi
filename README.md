# corsi

Repository inizializzato con una struttura minima per partire rapidamente.

## Obiettivo

Raccogliere materiali, esempi e codice relativi a corsi e percorsi formativi.

## Struttura attuale

- README.md: documentazione principale del progetto.

## Avvio rapido

1. Clona il repository.
2. Crea un branch per la tua modifica.
3. Aggiungi contenuti o codice.
4. Aggiorna questa documentazione se introduci nuove cartelle o strumenti.

## Convenzioni consigliate

- Mantieni nomi file e cartelle descrittivi e coerenti.
- Preferisci una cartella per ogni corso o argomento.
- Inserisci sempre un file README nelle nuove cartelle principali.

## Stato

Setup iniziale completato.
